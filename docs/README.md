# `Math formulas`

## *Circle*
- **Area**: S = πR²
- **Perimeter**: P = 2πR

## *Triangle*
- **Area**: S = a⋅h/2
- **Perimeter**: P = a + b + c

## *Rectangle*
- **Area**: S = a⋅b
- **Perimeter**: P = 2a + 2b

## *Square*
- **Area**: S = a²
- **Perimeter**: P = 4a
